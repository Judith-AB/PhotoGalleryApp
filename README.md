# PhotoGalleryApp 📸

This is a Photo Gallery App made with Flutter

## Features
- ✅ View an image  
- 📁 Create Albums  
- ✏️ Rename Albums  
- ❤️ Add images to Favorites  
- 🗑️ Delete images & Undo Delete  
- 🔄 Move images from one album to another
